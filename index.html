需求規格之足跡

1

專案建立

使用者粗略想法與利害關係人的發言。

Flow
Human(propose_rough_idea) → User(suggest_stakeholders) → Human(decision) → User(write_stakeholder_text) → project(stakeholders text)
Agent
User
Action
suggest_stakeholders、write_stakeholder_text
Input
Rough idea
Output
project.json
2

需求訪談

規劃需求擷取會議，讓各 agent 追問 User 以補足需求。

Flow
Mediator(plan_elicitation) → Agent(ask_user) → User(response) → elicitation_meeting
Agent
User
Action
plan_elicitation、ask_user、response
Input
project.json
Output
elicitation_meeting.json
3

需求分析

Analyst 分析出使用者的需求，並界定出需求範圍

Flow
project(stakeholders text) + elicitation_meeting → Analyst(analyze_scenario) → Analyst(analyze_requirements) → Analyst(generate_scope) → requirements / scope
Agent
Analyst
Action
analyze_scenario、analyze_requirements、generate_scope
Input
project.json、elicitation_meeting.json
Output
requirements.json、scope.json
4

需求衝突辨識

Analyst 辨識使用者需求是否有衝突，並產生衝突報告。

Flow
requirements → Analyst(analyze_conflicts) → result / conflict_report
Agent
Analyst
Action
analyze_conflicts
Input
requirements.json
Output
result.json、conflict_report_v0.json、conflict_report_v0.html
5

草稿建立

Analyst 將已知需求整理成初版草稿，作為後續討論基準。

Flow
project + requirements + scope → Analyst(create_draft) → draft_v0
Agent
Analyst
Action
create_draft
Input
project.json、requirements.json、scope.json
Output
draft_v0.html
6

提供回饋

Expert 檢查領域限制、風險與參考依據，整理成回饋。

Flow
draft_v0 → Expert(research_domain) → feedback
Agent
Expert
Action
research_domain
Input
draft_v0.html
Output
feedback.json

Modeler 建立系統模型，輔助理解流程、角色與邊界。

Flow
draft_v0 → Modeler(system_modeling) → system_models / models
Agent
Modeler
Action
system_modeling
Input
draft_v0.html
Output

、

Mediator 規劃正式會議，讓各 agent 討論議題並收束決議。

Flow
conflict_report_vN + draft_vN → Agent(propose_issues) → Mediator(plan_issues) → issues → Agent(agent_action) → requirements / conflict_report_vN → Agent(respond_issue) → Mediator(resolve_issue) → formal_meeting_rN / RN-MN
Agent
Analyst、Expert、Modeler、Mediator
Action
propose_issues、plan_issues、agent_action、discuss_conflict、respond_issue、resolve_issue

、
draft_vN.htmlv0v1
Output
issues.json、requirements.json、
conflict_report_vN.jsonv1
、
conflict_report_vN.htmlv1
、
formal_meeting_rN.jsonr1r2
、
RN-MN.htmlR1-M1.htmlR1-M2.htmlR2-M1.htmlR2-M2.htmlR2-M3.html

Mediator

、
